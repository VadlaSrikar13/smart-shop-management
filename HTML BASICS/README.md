# ShopOS - Smart Shop Management System

Complete full-stack app (frontend + Flask + SQLite) inside this folder.

## Setup
1. Open terminal in `HTML BASICS`.
2. Create/activate virtualenv:
   - `python -m venv venv`
   - `venv\Scripts\activate` (Windows)
3. Install packages:
   - `pip install -r requirements.txt`
4. Start the app:
   - `python app.py`
5. Open browser: `http://localhost:5000`

## Login
- Customer: `user` / `user123`
- Admin: `admin` / `admin123`

## Features
- Customer: product listing/search/category, add-to-cart, quantity, checkout
- Admin: dashboard stats, product CRUD, orders, low stock alerts
- Invoice & order history
